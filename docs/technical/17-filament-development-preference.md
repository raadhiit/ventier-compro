# 17. Filament Development Preference

- Prefer native Filament features before custom components.
- Group form fields logically.
- Add validation and helper text where administrators may misunderstand a field.
- Provide useful search and filters in tables.
- Enable reordering only for records that have a meaningful order.
- Show upload previews.
- Require confirmation for destructive actions.
- Add bulk actions only when they provide operational value.
- Do not expose technical fields that confuse content administrators.
