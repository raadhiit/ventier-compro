# 24. Out of Scope for the Initial Version

The following are excluded unless approved as new requirements:

- Shopping cart.
- Online checkout.
- Payment gateway.
- Customer accounts.
- Order management.
- Inventory management.
- Marketplace integration.
- Multi-vendor support.
- Arbitrary drag-and-drop page builder.
- Complex marketing automation.
- Multilingual support.
- Mobile application.
- Microservices.
- Custom real-time chat.

A WhatsApp deep link is allowed and is more appropriate for the initial release than building a custom live-chat system.
