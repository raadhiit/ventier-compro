# 05. Main Site Architecture

The public website has four fixed primary navigation items:

1. **Home**
2. **Product**
3. **About Us**
4. **Contact**

The primary navigation structure is fixed for the initial release. Administrators must not be allowed to create arbitrary navigation items because this introduces broken navigation risks and expands the CMS into a page builder.

Administrators may manage content, selected labels, predefined section order, and section visibility. They must not freely change the core public layout.
