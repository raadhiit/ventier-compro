# 02. Product Overview

Ventier is a brand that sells car carpet products.

The website serves two primary functions:

1. **Company profile:** introduce the brand and build credibility.
2. **Marketing website:** present products, explain product benefits, and direct visitors to the intended contact or sales channel.

The website must be informative, responsive, easy to use, fast, and visually interactive without reducing stability, readability, or accessibility.

A Filament-based CMS must allow the Ventier team to manage core content without changing source code.
