# 16. Livewire Usage Preference

Use Livewire only where it provides clear value.

## Appropriate Uses

- Product search.
- Product filters.
- Pagination or load more.
- Contact form submission.
- Dynamic product-gallery state.
- Custom admin interactions when native Filament behavior is insufficient.

## Inappropriate Uses

- Static headings or paragraphs.
- Static navigation without meaningful state.
- One oversized component controlling an entire page.
- Visual effects that CSS can handle more reliably.

Components must remain focused, small enough to understand, and free of unnecessary state.
