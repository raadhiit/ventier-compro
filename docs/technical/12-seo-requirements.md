# 12. SEO Requirements

SEO is part of the application structure, not a final-stage add-on.

## Minimum Features

- Semantic heading hierarchy.
- Unique title and meta description for each product and article.
- Canonical URLs.
- Open Graph metadata.
- Social-sharing images.
- XML sitemap.
- Robots configuration.
- Human-readable slugs.
- Structured data when relevant.
- Image alt text.
- Breadcrumbs on detail pages when useful.
- Redirect or safe handling when a slug changes.

## Admin Editing

Administrators may edit:

- SEO title.
- SEO description.
- Social image when needed.

When SEO fields are empty, the system must generate safe fallbacks from the content name or default site settings.
