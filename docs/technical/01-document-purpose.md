# 01. Document Purpose

**Project:** Ventier Company Profile and CMS  
**Version:** 1.1  
**Status:** Primary development reference  
**Last updated:** 15 July 2026

This document set is the primary reference for developing the Ventier company profile website and CMS.

All major decisions related to design, features, database structure, public user experience, and admin-panel behavior must follow these references. Any major deviation must have a clear business or technical justification and must be recorded as a requirement change.

These references define project boundaries. The system must not gradually turn into a general-purpose page builder, marketplace, e-commerce platform, or enterprise system without an explicitly approved requirement.
